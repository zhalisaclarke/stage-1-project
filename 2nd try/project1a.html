Lesson 1: The Basics

The Web - Demystified

The World Wide Web is a system of Internet servers that support specially formatted documents. The documents are formatted in a markup language called HTML (HyperText Markup Language) that supports links to other documents, as well as graphics, audio, and video files. This means you can jump from one document to another simply by clicking on hot spots.

Tips to Remember

Closing tags - If you leave off the close tag, everything after the opening tag will be bolded/italized or whatever the command was

Block vs In line - italics, bold, and the "break" tags are all in line; however paragraph is block, as well as div.

HTML vs CSS

HTML and CSS are both "languages. "HTML controls the "structure" of a web page. CSS controls the "style" of a page (how it looks). HTML will refer to corresponding CSS file, and it's important to save them in the same folder.

Lesson 3: Adding CSS Style to HTML

Avoiding Repetition

In Lesson 3 I learned a lot about CSS, and how to add styling elements to HTML while reducing repetition in my code. Luckily there are CSS reference tools because there is no way I could remember all of this. I also learned how to use developer tools to test CSS changes and get real time feedback without changing the code itself.

While you could put styling code directly in your HTML, it leads to a lot of repetition, and risk of error, so for larger files it’s best to create two separate CSS and HTML files. And, once you’re happy with the HTML, you can play around with the CSS to make it look exactly as you want. Code, test and refine. Rinse repeat.
